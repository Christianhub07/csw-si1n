neste exemplo vou inserir o comando 'valores': [10, 12, 15, 14, 100, 12, 14, 13, 12, 12] que é um comando python

Inserindo um longo comando em python: 


            import numpy as np
import pandas as pd
from scipy import stats

# Exemplo de dados
data = {
    'valores': [10, 12, 15, 14, 100, 12, 14, 13, 12, 12]
}

# Criando um DataFrame
df = pd.DataFrame(data)

# Calculando o Z-Score
z_scores = np.abs(stats.zscore(df['valores']))

# Definindo um limite para o Z-Score
limite = 3

# Remover outliers (onde o Z-Score é maior que o limite)
df_sem_outliers = df[z_scores < limite]

print("Dados sem outliers:")
print(df_sem_outliers)
